MongoInt64::__construct

(PECL mongo >= 1.0.9)

MongoInt64::__construct — 创建一个新的 64 位 integer。

说明

public MongoInt64::__construct ( string $value )

创建具有指定值的新的 64 位数字。

参数

value

一个数字。

返回值

返回一个新的 integer。
