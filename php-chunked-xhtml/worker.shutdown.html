Worker::shutdown

(PECL pthreads >= 2.0.0)

Worker::shutdown — Synchronization

说明

public boolean Worker::shutdown ( void )

Shuts down the Worker after executing all the objects previously stacked

参数

此函数没有参数。

返回值

范例

Example #1 Shutdown the referenced Worker

<?php
$my = new Worker();
$my->start();
/* ... */
var_dump($my->shutdown());
/* ... */
?>

以上例程会输出：

bool(true)
