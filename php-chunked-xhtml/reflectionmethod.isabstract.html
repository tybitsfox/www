ReflectionMethod::isAbstract

(PHP 5)

ReflectionMethod::isAbstract — 判断方法是否是抽象方法

说明

public bool ReflectionMethod::isAbstract ( void )

判断方法是否是抽象方法

参数

此函数没有参数。

返回值

如果是抽象方法返回 TRUE，否则返回 FALSE

参见

  • ReflectionMethod::getDeclaringClass() - 获取反射函数调用参数的类表达
