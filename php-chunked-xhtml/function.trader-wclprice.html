trader_wclprice

(PECL trader >= 0.2.0)

trader_wclprice — Weighted Close Price

说明

array trader_wclprice ( array $high , array $low , array $close )

参数

high

高价，浮点数数组。

low

低价，浮点数数组。

close

收盘价，浮点数数组。

返回值

Returns an array with calculated data or false on failure.
