mysqli::thread_safe

mysqli_thread_safe

(PHP 5)

mysqli::thread_safe -- mysqli_thread_safe — 返回是否是线程安全的

说明

过程化风格

bool mysqli_thread_safe ( void )

告知本数据库客户端库是否编译为线程安全的。 Tells whether the client library is compiled as thread-safe.

返回值

如果是的话，返回TRUE，否则返回FALSE。
