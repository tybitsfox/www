Counter::getNamed

Counter::getNamed — 取回一个名称已存在的计数器。

说明

static Counter Counter::getNamed ( string $name )

Counter::getNamed() 通过名称查询一个已存在的计数器，不存在则返回NULL。这是静态函数。

参数

name
要查询的计数器名称。

返回值

Counter::getNamed() 通过名称查询一个已存在的计数器，不存在则返回NULL。
