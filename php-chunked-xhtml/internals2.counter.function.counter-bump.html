counter_bump

counter_bump — 修改简单计数器的当前值。

说明

void counter_bump ( integer $offset )

counter_bump() 修改使用简单接口的计数器的当前值。

参数

offset
计数器值的变化量。可为负数。

参见

  • counter_get() - 获取简单计数器的当前值。
  • counter_reset() - 重置简单计数器的当前值。
