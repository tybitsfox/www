MongoId::getTimestamp

(PECL mongo >= 1.0.1)

MongoId::getTimestamp — 获取新纪元时间到 id 创建时的秒数。

说明

public int MongoId::getTimestamp ( void )

返回 id 创建时运行 time() 同样的东西。

参数

此函数没有参数。

返回值

获取新纪元时间到 id 创建时的秒数。 仅储存了四个字节的时间戳，所以 MongoDate 是储存更精准、更广范围时间的更佳选择。
