LuaClosure::__invoke

LuaClosure::__invoke — 调用luaclosure

说明

public void LuaClosure::__invoke ( mixed $arg [, mixed $... ] )

Warning

本函数还未编写文档，仅有参数列表。

参数

arg

...

返回值

范例

Example #1 LuaClosure::__invoke()示例

<?php
$lua = new Lua();
$closure = $lua->eval(<<<CODE
    return (function ()
        print("hello world")
    end)
CODE
);

$lua->call($closure);
/* after PHP 5.3 */
$closure();
?>

以上例程会输出：

hello worldhello world
