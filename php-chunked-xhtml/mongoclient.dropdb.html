MongoClient::dropDB

(PECL mongo >=1.3.0)

MongoClient::dropDB — 删除一个数据库 [已废弃]

说明

public array MongoClient::dropDB ( mixed $db )
Warning

已废弃

请使用 MongoDB::drop() 作为替代。

参数

db

要函数的数据库。可以是一个 MongoDB对象，或者是数据库名。

返回值

返回数据库的回应。
