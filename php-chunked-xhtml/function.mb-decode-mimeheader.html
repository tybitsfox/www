mb_decode_mimeheader

(PHP 4 >= 4.0.6, PHP 5)

mb_decode_mimeheader — 解码 MIME 头字段中的字符串

说明

string mb_decode_mimeheader ( string $str )

解码 MIME 头中编码过的字符串 str。

参数

str

要解码的 string。

返回值

以内部（internal）字符编码解码的 string。

参见

  • mb_encode_mimeheader() - 为 MIME 头编码字符串
