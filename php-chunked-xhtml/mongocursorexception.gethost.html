MongoCursorException::getHost

(PECL mongo >= 1.0.0)

MongoCursorException::getHost — 遇到该错误的服务器的 hostname

说明

public string MongoCursorException::getHost ( void )

返回发送查询到该服务器的 hostname。

参数

此函数没有参数。

返回值

返回 hostname，如果 hostname 是未知的则是 NULL。
