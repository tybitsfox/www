OAuth::disableDebug

(PECL OAuth >= 0.99.3)

OAuth::disableDebug — 关闭详细的调试

说明

public bool OAuth::disableDebug ( void )

关闭详细的请求信息（默认为关闭）。或者，将 调试 属性设置为 FALSE 值来关闭调试。

参数

此函数没有参数。

返回值

TRUE

更新日志

版本 说明
0.99.8 增加相关 调试 属性。
