Imagick::displayImage

(PECL imagick 2.0.0)

Imagick::displayImage — Displays an image

说明

bool Imagick::displayImage ( string $servername )

This method displays an image on a X server.

参数

servername

The X server name

返回值

成功时返回 TRUE。

错误／异常

错误时抛出 ImagickException。
