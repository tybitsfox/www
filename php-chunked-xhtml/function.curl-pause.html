curl_pause

(PHP 5 >= 5.5.0)

curl_pause — Pause and unpause a connection

说明

int curl_pause ( resource $ch , int $bitmask )

Warning

本函数还未编写文档，仅有参数列表。

参数

ch

由 curl_init() 返回的 cURL 句柄。

bitmask

One of CURLPAUSE_* constants.

返回值

Returns an error code (CURLE_OK for no error).
