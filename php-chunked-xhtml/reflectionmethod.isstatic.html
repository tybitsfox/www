ReflectionMethod::isStatic

(PHP 5)

ReflectionMethod::isStatic — 判断方法是否是静态方法

说明

public bool ReflectionMethod::isStatic ( void )

判断方法是否是静态方法

参数

此函数没有参数。

返回值

静态方法返回 TRUE，否则返回 FALSE

参见

  • ReflectionMethod::isFinal() - 判断方法是否定义 final
