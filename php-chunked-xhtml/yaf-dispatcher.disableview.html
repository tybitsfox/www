Yaf_Dispatcher::disableView

(Yaf >=1.0.0)

Yaf_Dispatcher::disableView — 关闭自动渲染

说明

public bool Yaf_Dispatcher::disableView ( void )

在一些用户自己会输出信息的情况下使用。

Note:

你可以在一个action中仅仅返回FALSE来阻止当前action对应视图的自动渲染

参数

此函数没有参数。

返回值
