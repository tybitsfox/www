Closure::__construct

(PHP 5 >= 5.3.0)

Closure::__construct — 用于禁止实例化的构造函数

说明

Closure::__construct ( void )

这个方法仅用于禁止实例化一个 Closure 类的对象。这个类的对象的创建方法写在 匿名函数 页。

参数

此函数没有参数。

返回值

没有返回值，它只是简单的触发一个错误 （类型是 E_RECOVERABLE_ERROR）。
