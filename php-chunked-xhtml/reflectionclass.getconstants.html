ReflectionClass::getConstants

(PHP 5)

ReflectionClass::getConstants — 获取一组常量

说明

public array ReflectionClass::getConstants ( void )

获取一个类中所有已定义的常量。

Warning

本函数还未编写文档，仅有参数列表。

参数

此函数没有参数。

返回值

常量的数组，常量名是数组的键，常量的值是数组的值。

参见

  • ReflectionClass::getConstant() - 获取定义过的一个常量
