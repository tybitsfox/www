MongoTimestamp::__toString

(PECL mongo >= 1.0.1)

MongoTimestamp::__toString — 返回时间戳的字符串表示形式

说明

public string MongoTimestamp::__toString ( void )

返回时间戳里的秒数字段。

参数

此函数没有参数。

返回值

该时间戳自新纪元时间以来的秒数。
