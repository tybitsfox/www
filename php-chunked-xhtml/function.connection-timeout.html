connection_timeout

(PHP 4 <= 4.0.4)

connection_timeout — 检查脚本是否已超时

说明

int connection_timeout ( void )

检查脚本是否已超时。

返回值

如果脚本已超时返回 1，否则返回 0。

注释

Warning

已废弃

该函数已被废弃，从PHP4.0.5起不再存在。

参见

  • connection_status() - 返回连接的状态位
  • Connection Handling 关于PHP连接处理的完整描述。
