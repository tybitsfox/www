Threaded::pop

(PECL pthreads >= 2.0.0)

Threaded::pop — Manipulation

说明

public boolean Threaded::pop ( void )

弹出对象属性表中的最后一项数据

返回值

对象属性表中最后一项数据

范例

Example #1 弹出对象属性表中的最后一项数据

<?php
$safe = new Threaded();

while (count($safe) < 10)
    $safe[] = count($safe);

var_dump($safe->pop());
?>

以上例程会输出：

int(9)
