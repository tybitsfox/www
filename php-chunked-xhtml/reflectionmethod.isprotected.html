ReflectionMethod::isProtected

(PHP 5)

ReflectionMethod::isProtected — 判断方法是否是保护方法 (protected)

说明

public bool ReflectionMethod::isProtected ( void )

判断方法是否是保护方法

参数

此函数没有参数。

返回值

保护方法返回 TRUE，否则返回 FALSE

参见

  • ReflectionMethod::isPrivate() - 判断方法是否是私有方法
