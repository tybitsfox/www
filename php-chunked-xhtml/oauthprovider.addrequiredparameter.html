OAuthProvider::addRequiredParameter

(PECL OAuth >= 1.0.0)

OAuthProvider::addRequiredParameter — 添加必需的参数

说明

final public bool OAuthProvider::addRequiredParameter ( string $req_params )

添加必需的 oauth 提供者参数。

Warning

本函数还未编写文档，仅有参数列表。

参数

req_params

必需的参数。

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。

参见

  • OAuthProvider::removeRequiredParameter() - 移除一个必需的参数
