需求

此扩展需要 libxml PHP 扩展。这表示需要使用 --enable-libxml ，尽管这将隐式完成因为 libxml 是缺省开启的。

This extension uses Sablotron and expat, which can both be found at » http://sablotron.sourceforge.net/. Binaries are provided as well as source.

Enable by using the --with-xslt option with PHP 4.
