Counter::setCounterClass

Counter::setCounterClass — 设置由 Counter::getNamed() 返回的计数器类。

说明

static void Counter::setCounterClass ( string $name )

Counter::setCounterClass() 改变由 Counter::getNamed() 返回的对象的类。要设置的类可以没有 pulic 的构造器，但必须是 Counter 的子类。如果不满足这些条件，则会引发致命错误。这是静态函数。

参数

name
要使用的类名。
