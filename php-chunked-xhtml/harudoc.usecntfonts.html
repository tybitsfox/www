HaruDoc::useCNTFonts

(PECL haru >= 0.0.1)

HaruDoc::useCNTFonts — Enable builtin Chinese traditional fonts

说明

bool HaruDoc::useCNTFonts ( void )

Enables builtin Chinese traditional fonts.

参数

此函数没有参数。

返回值

Returns TRUE on success.

错误／异常

发生错误时抛出 HaruException 异常。

参见
