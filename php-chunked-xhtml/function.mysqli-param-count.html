mysqli_param_count

(PHP 5 < 5.4.0)

mysqli_param_count — mysqli_stmt_param_count()的别名

说明

这个函数是mysqli_stmt_param_count()的一个别名。

注释

Note:

mysqli_param_count()已经被废弃并且将会被移除。

参见

  • mysqli_stmt_param_count() - Returns the number of parameter for the given statement
