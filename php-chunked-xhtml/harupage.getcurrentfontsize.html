HaruPage::getCurrentFontSize

(PECL haru >= 0.0.1)

HaruPage::getCurrentFontSize — Get the current font size

说明

float HaruPage::getCurrentFontSize ( void )

Get the current font size.

参数

此函数没有参数。

返回值

Returns the current font size.

错误／异常

发生错误时抛出 HaruException 异常。
