ArrayAccess::offsetUnset

(PHP 5 >= 5.0.0)

ArrayAccess::offsetUnset — 复位一个偏移位置的值

说明

abstract public void ArrayAccess::offsetUnset ( mixed $offset )

复位一个偏移位置的值。

Note:

当使用 (unset) 进行类型转换时，该方法不会被调用。

参数

offset

待复位的偏移位置。

返回值

没有返回值。
