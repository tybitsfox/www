Threaded::unlock

(PECL pthreads >= 2.0.0)

Threaded::unlock — Synchronization

说明

public boolean Threaded::unlock ( void )

解锁对象

参数

此函数没有参数。

返回值

布尔值，表示操作是否成功

范例

Example #1 给对象属性表加锁

<?php
class My extends Thread {
    public function run() {
        var_dump($this->lock());
        /** 其他线程无法执行读/写操作 **/
        var_dump($this->unlock());
        /** 其他线程可以执行读/写操作 */
    }
}
$my = new My();
$my->start();
?>

以上例程会输出：

bool(true)
bool(true)
