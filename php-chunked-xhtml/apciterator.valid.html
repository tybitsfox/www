APCIterator::valid

(PECL apc >= 3.1.1)

APCIterator::valid — 检查当前位置是否有效

说明

public void APCIterator::valid ( void )

检查当前迭代器的位置是否有效。

参数

此函数没有参数。

返回值

如果当前迭代器的位置有效返回 TRUE ，反之则返回 FALSE。

参见

  • APCIterator::current() - 获取当前项
  • Iterator::valid() - 检查当前位置是否有效
