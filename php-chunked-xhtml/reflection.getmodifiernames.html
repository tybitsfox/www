Reflection::getModifierNames

(PHP 5)

Reflection::getModifierNames — 获取修饰符的名称

说明

public static array Reflection::getModifierNames ( int $modifiers )

获取修饰符的名称。

Warning

本函数还未编写文档，仅有参数列表。

参数

modifiers

根据数字的值获取修饰符。

返回值

修饰符名称的一个数组。
