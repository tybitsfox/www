SphinxClient::close

(PECL sphinx >= 1.0.3)

SphinxClient::close — 关闭先前打开的持久连接

说明

public bool SphinxClient::close ( void )

关闭先前打开的持久连接.

参数

此函数没有参数。

更新日志

PECL/sphinx 版本 说明
1.0.3 新增函数 SphinxClient::close(), 仅在编译时使用 libsphinxclient >= 0.9.9 有效.

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。
