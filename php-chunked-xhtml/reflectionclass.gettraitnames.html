ReflectionClass::getTraitNames

(PHP >= 5.4.0)

ReflectionClass::getTraitNames — 返回这个类所使用 traits 的名称的数组

说明

public array ReflectionClass::getTraitNames ( void )

Warning

本函数还未编写文档，仅有参数列表。

参数

此函数没有参数。

返回值

返回的数组的值包含了 trait 的名称。 出现错误的情况下返回 NULL。
