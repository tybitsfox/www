OAuth::disableSSLChecks

(PECL OAuth >= 0.99.5)

OAuth::disableSSLChecks — 关闭 SSL 检查

说明

public bool OAuth::disableSSLChecks ( void )

关闭通常的 SSL 对等证书和主机检查，但不用于生产环境。或者，设置 sslChecks 成员为 FALSE 来关闭 SSL 检查。

参数

此函数没有参数。

返回值

TRUE

更新日志

版本 说明
0.99.8 增加了 sslChecks 成员
