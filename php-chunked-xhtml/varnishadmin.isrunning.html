VarnishAdmin::isRunning

(PECL varnish >= 0.3)

VarnishAdmin::isRunning — Check if the varnish slave process is currently running

说明

public bool VarnishAdmin::isRunning ( void )

参数

此函数没有参数。

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。
