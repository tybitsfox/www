APCIterator::rewind

(PECL apc >= 3.1.1)

APCIterator::rewind — 倒退迭代器

说明

public void APCIterator::rewind ( void )

倒退迭代器到第一项。

参数

此函数没有参数。

返回值

没有返回值。

参见

  • APCIterator::next() - 移到下一项
  • Iterator::next() - 向前移动到下一个元素
