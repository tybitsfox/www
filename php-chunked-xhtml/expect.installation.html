安装

此 » PECL 扩展未与 PHP 捆绑。 安装此 PECL 扩展相关的信息可在手册中标题为 PECL 扩展的安装章节中找到。更多信息如新的发行版本、下载、源文件、 维护人员信息及变更日志等，都在此处： » http://pecl.php.net/package/expect.

在 PHP 4 中，此 PECL 扩展源代码可在 PHP 源代码的 ext/ 目录下或上面的 PECL链接中找到。 In order to use these functions you must compile PHP with expect support by using the --with-expect[=DIR] configure option.

Windows users will enable php_expect.dll inside of php.ini in order to use these functions. 在 PHP 4 中，此 DLL 位于 PHP Windows 二进制下载中的 extensions/ 目录。 PECL 扩展的 DLL 当前不可用。参见 在 Windows 上构建章节。
