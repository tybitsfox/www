ps_setflat

(PECL ps >= 1.1.0)

ps_setflat — Sets flatness

说明

bool ps_setflat ( resource $psdoc , float $value )

Warning

本函数还未编写文档，仅有参数列表。

参数

psdoc

Resource identifier of the postscript file as returned by ps_new().

value

The value must be between 0.2 and 1.

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。
