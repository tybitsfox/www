OCI-Lob::import

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Lob::import — Imports file data to the LOB

说明

bool OCI-Lob::import ( string $filename )

Writes data from the filename in to the current position of large object.

参数

filename

Path to the file.

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。

参见

  • OCI-Lob::export
  • OCI-Lob::write
