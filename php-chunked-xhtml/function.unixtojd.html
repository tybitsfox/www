unixtojd

(PHP 4, PHP 5)

unixtojd — 转变Unix时间戳为Julian Day计数

说明

int unixtojd ([ int $timestamp = time() ] )

根据指定的Unix时间戳timestamp，返回Julian天数。如果没有指定时间戳则返回当前日期的天数。

参数

timestamp

一个用于转变的时间戳。

返回值

一个julian天数。

参见

  • jdtounix() - 转变Julian Day计数为一个Unix时间戳
