ReflectionClass::getTraitAliases

(PHP >= 5.4.0)

ReflectionClass::getTraitAliases — 返回 trait 别名的一个数组

说明

public array ReflectionClass::getTraitAliases ( void )

Warning

本函数还未编写文档，仅有参数列表。

参数

此函数没有参数。

返回值

返回了一个数组，新的方法名位于键中，原始名称（格式是 "TraitName::original"）位于数组的值中。 出现一个错误的情况下返回 NULL。
