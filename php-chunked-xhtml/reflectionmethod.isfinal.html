ReflectionMethod::isFinal

(PHP 5)

ReflectionMethod::isFinal — 判断方法是否定义 final

说明

public bool ReflectionMethod::isFinal ( void )

判断方法是否定义 final

参数

此函数没有参数。

返回值

如果是 final 返回 TRUE，否则返回 FALSE

参见

  • ReflectionMethod::isStatic() - 判断方法是否是静态方法
