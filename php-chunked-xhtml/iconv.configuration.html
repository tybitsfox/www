运行时配置

这些函数的行为受 php.ini 中的设置影响。

Iconv 配置选项
名字 默认 可修改范围 更新日志
iconv.input_encoding "ISO-8859-1" PHP_INI_ALL 自 PHP 4.0.5 起有效。
iconv.output_encoding "ISO-8859-1" PHP_INI_ALL 自 PHP 4.0.5 起有效。
iconv.internal_encoding "ISO-8859-1" PHP_INI_ALL 自 PHP 4.0.5 起有效。

这是配置指令的简短说明。

Warning

有些系统（比如 IBM AIX） 使用 "ISO8859-1" 来代替 "ISO-8859-1"，所以在配置选项和函数参数中必须使用这个值。

iconv.input_encoding string

目前没用使用到这个选项。

iconv.output_encoding string

iconv.internal_encoding string
