MongoBinData::__toString

(PECL mongo >= 0.8.1)

MongoBinData::__toString — 二进制数据对象的字符串表达形式。

说明

public string MongoBinData::__toString ( void )

参数

此函数没有参数。

返回值

返回字符串 "<Mongo Binary Data>"。要获取 MongoBinData 的内容，使用 bin 字段。
