hw_Free_Document

(PHP 4)

hw_Free_Document — Frees hw_document

说明

bool hw_free_document ( int $hw_document )

Frees the memory occupied by the Hyperwave document.

参数

hw_document

The document identifier.

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。
