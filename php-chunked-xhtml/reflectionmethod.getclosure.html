ReflectionMethod::getClosure

(PHP >= 5.4.0)

ReflectionMethod::getClosure — 返回一个动态建立的方法调用接口，译者注：可以使用这个返回值直接调用非公开方法。

说明

public Closure ReflectionMethod::getClosure ( object $object )

Warning

本函数还未编写文档，仅有参数列表。

参数

object

不可以使用静态方法，需要其他类型的方法

返回值

返回 Closure 如果产生任何错误返回 NULL
