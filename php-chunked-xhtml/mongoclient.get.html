MongoClient::__get

(PECL mongo >=1.3.0)

MongoClient::__get — 取得一个数据库

说明

public MongoDB MongoClient::__get ( string $dbname )

这是获取一个数据库最简洁的方法。 如果数据库名称包含任意特殊字符，需要使用 MongoClient::selectDB() 方法；但是大多数情况下这个方法就可以。

<?php

$mongo = new MongoClient();

// 以下两行是一样的
$db = $mongo->selectDB("foo");
$db = $mongo->foo;

?>

参数

dbname

数据库名

返回值

返回一个新的 db 对象。

错误／异常

如果数据库名无效，将会抛出异常。
