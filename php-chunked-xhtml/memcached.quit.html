Memcached::quit

(PECL memcached >= 2.0.0)

Memcached::quit — Close any open connections

说明

public bool Memcached::quit ( void )

Memcached::quit() closes any open connections to the memcache servers.

参数

此函数没有参数。

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。
