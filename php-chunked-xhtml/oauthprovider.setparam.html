OAuthProvider::setParam

(PECL OAuth >= 1.0.0)

OAuthProvider::setParam — 设置一个参数

说明

final public bool OAuthProvider::setParam ( string $param_key [, mixed $param_val ] )

设置一个参数。

Warning

本函数还未编写文档，仅有参数列表。

参数

param_key

参数的 key 。

param_val

参数的值，此为可选项。

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。

参见

  • OAuthProvider::addRequiredParameter() - 添加必需的参数
