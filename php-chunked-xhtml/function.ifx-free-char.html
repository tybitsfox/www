ifx_free_char

(PHP 4, PHP <=5.2.0)

ifx_free_char — Deletes the char object

说明

bool ifx_free_char ( int $bid )

Deletes the charobject for the given char object-id.

参数

bid

The char object id.

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。

参见

  • ifx_create_char() - Creates an char object
