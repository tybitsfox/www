xslt_free

(PHP 4 >= 4.0.3)

xslt_free — Free XSLT processor

说明

void xslt_free ( resource $xh )

Free the XSLT processor identified by the given handle.

参数

xh

XSLT 处理器连接标识符，由 xslt_create() 创建。

返回值

没有返回值。

参见

  • xslt_create() - Create a new XSLT processor
