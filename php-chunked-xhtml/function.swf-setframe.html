swf_setframe

(PHP 4)

swf_setframe — Switch to a specified frame

说明

void swf_setframe ( int $framenumber )

Changes the active frame to the specified on.

参数

framenumber

The frame number to be set.

返回值

没有返回值。

参见

  • swf_getframe() - Get the frame number of the current frame
