Gmagick::blurimage — Adds blur filter to image

说明

public Gmagick Gmagick::blurimage ( float $radius , float $sigma [, int $channel ] )

Adds blur filter to image.

参数

radius

Blur radius

sigma

Standard deviation

返回值
