HaruDoc::useKRFonts

(PECL haru >= 0.0.1)

HaruDoc::useKRFonts — Enable builtin Korean fonts

说明

bool HaruDoc::useKRFonts ( void )

Enables builtin Korean fonts.

参数

此函数没有参数。

返回值

Returns TRUE on success.

错误／异常

发生错误时抛出 HaruException 异常。

参见

  • HaruDoc::useKREncodings() - Enable Korean encodings
