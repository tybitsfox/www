Counter::resetValue

Counter::resetValue — 重置计数器的当前值。

说明

void Counter::resetValue ( void )

Counter::resetValue() 将计数器的当前值重置为其初始值。

参见

  • Counter::getValue() - 获取计数器的当前值。
