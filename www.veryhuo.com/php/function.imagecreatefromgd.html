imagecreatefromgd

(PHP 4 >= 4.1.0)

imagecreatefromgd -- 从 GD 文件或 URL 新建一图像

说明

resource imagecreatefromgd ( string filename)

警告

该函数目前尚无参考文档；仅参数列表可用。

提示: 如果“fopen wrappers”已经被激活，则您在使用该函数的时候，可以把 URL 作为文件名来使用。请参阅 fopen() 获取怎样指定文件名的详细信息以及支持的 URL 协议的列表附录 J。

警告

Windows 版本的 PHP 在 4.3.0 版之前不支持该函数的远程文件访问，即使 allow_url_fopen 选项已被激活。
