ora_do

(PHP 3, PHP 4 )

ora_do -- 分析，执行，获取

描述

resource ora_do ( resource conn, string query)

ora_do()一次性执行函数 ora_parse()， ora_exec() 和 ora_fetch() 所要执行的功能。它将分析和执行一条语句，并取得结果中的第一行。

该函数执行成功则返回一个游标索引，失败则返回 FALSE。 错误的细节能够使用 ora_error() 和 ora_errorcode() 函数取得。

参见 ora_parse()，ora_exec()， 和 ora_fetch()。
