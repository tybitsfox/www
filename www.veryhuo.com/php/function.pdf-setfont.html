pdf_setfont

(PHP 4 >= 4.0.5)

pdf_setfont -- Set the current font

Description

bool pdf_setfont ( resource pdfdoc, int font, float size)

Set the current font in the given size, using a font handle returned by pdf_findfont(). 如果成功则返回 TRUE，失败则返回 FALSE。

See also pdf_findfont().
