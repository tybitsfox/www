fgetc

(PHP 3, PHP 4 )

fgetc -- 从文件指针中读取字符

说明

string fgetc ( resource handle)

返回一个包含有一个字符的字符串，该字符从 handle 指向的文件中得到。碰到 EOF 则返回 FALSE。

文件指针必须有效，并且是通过 fopen()，popen() 或 fsockopen() 成功打开的。

警告

这个函数可能返回 Boolean FALSE，也可能返回一个与 FALSE 相等地非 Boolean 值，比如 0 或者 "" 。请参阅 Booleans 的有关章节以获取更多信息。请使用 === 操作符 来测试该函数的返回值。

例子 1. fgetc() 例子

<?php
$fp = fopen('somefile.txt', 'r');
if (!$fp) {
    echo 'Could not open file somefile.txt';
}
while (false !== ($char = fgetc($fp))) {
    echo "$char\n";
}
?>

注: 该函数适用于二进制对象！

参见 fread()，fopen()，popen()，fsockopen() 和 fgets()。
