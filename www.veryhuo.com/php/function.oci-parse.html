oci_parse -- Parses a query and returns an Oracle statement identifier

Description

resource oci_parse ( resource connection, string query)

oci_parse() parses the query using connection and returns the statement identifier if the query is valid or FALSE if not. The query can be any valid SQL statement or PL/SQL block.

oci_parse() returns FALSE on error.
