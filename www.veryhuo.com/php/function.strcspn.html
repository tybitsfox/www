strcspn

(PHP 3>= 3.0.3, PHP 4 )

strcspn --  Find length of initial segment not matching mask

Description

int strcspn ( string str1, string str2)

Returns the length of the initial segment of str1 which does not contain any of the characters in str2.

See also strspn().
