Oreilly Website Pro 服务器

本章节的内容和提示仅限于将 PHP 安装到 Oreilly Website Pro 服务器上.

在 Windows 系统下运行的 Oreilly Website Pro 2.5 以及更新的版本

本文将主要阐述如何使 PHP 使用 CGI 方式或 ISAPI 模块方式与 Windows 系统的下 Oreilly Website Pro 一起工作。

  • 编辑服务器属性并且选择“Mapping”标签。

  • 在列表里选择“Associations”并且输入扩展名（.php）以及指定 CGI 可执行文件（.EXE）的路径（例如：c:\php\php.exe）或者是 ISAPI 动态链接库文件（.DLL）的路径（例如：c:\php\sapi\php4isapi.dll）。

  • 选择“Content Types”添加与“Associations”列表中相同的扩展名（.php）并且输入 content type，如果你是用 CGI 可执行文件请输入“wwwserver/shellcgi”，如果使用了 ISAPI 模块则请输入“wwwserver/isapi”（不包含引号）。
