ldap_delete

(PHP 3, PHP 4 )

ldap_delete -- Delete an entry from a directory

Description

bool ldap_delete ( resource link_identifier, string dn)

如果成功则返回 TRUE，失败则返回 FALSE。

ldap_delete() function delete a particular entry in LDAP directory specified by dn.
