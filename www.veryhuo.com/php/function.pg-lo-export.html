pg_lo_export

(PHP 4 >= 4.2.0)

pg_lo_export -- 将大型对象导出到文件

说明

bool pg_lo_export ( int oid, string pathname [, resource connection])

oid 参数指定了要导出的大型对象的 oid，pathname 参数则指明了文件名。如果成功则返回 TRUE，失败则返回 FALSE。

要使用大型对象（lo）接口，需要将其放置在事务块中。

注: 本函数以前的名字为 pg_loexport()。

参见 pg_lo_import()。
