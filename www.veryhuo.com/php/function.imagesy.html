imagesy

(PHP 3, PHP 4 )

imagesy -- 取得图像高度

说明

int imagesy ( resource image)

imagesy() 返回 image 所代表的图像的高度。

例子 1. 使用 imagesy()

<?php

// create a 300*200 image
$img = imagecreate(300, 200);

echo imagesy($img); // 200

?>

参见 imagecreate()，getimagesize() 和 imagesx()。
