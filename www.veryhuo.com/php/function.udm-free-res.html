udm_free_res

(PHP 4 >= 4.0.5)

udm_free_res -- Free mnoGoSearch result

Description

bool udm_free_res ( resource res)

如果成功则返回 TRUE，失败则返回 FALSE。

res - a link to result identifier, received after call to udm_find().

Freeing up memory allocated for results.
