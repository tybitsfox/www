fclose

(PHP 3, PHP 4 )

fclose -- 关闭一个已打开的文件指针

说明

bool fclose ( resource handle)

将 handle 指向的文件关闭。

如果成功则返回 TRUE，失败则返回 FALSE。

文件指针必须有效，并且是通过 fopen() 或 fsockopen() 成功打开的。

例子 1. 一个简单 fclose() 例子

<?php
  $handle = fopen('somefile.txt', 'r');
  fclose($handle);
?>
