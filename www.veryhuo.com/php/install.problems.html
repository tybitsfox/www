阅读 FAQ

有些问题比其它问题更普通。最普通的问题被列在本手册的 PHP 常见问题 中。

其它问题

如果问题依然存在，在 PHP 安装邮件列表里的某个人也许能够帮助你。 你应当在提问之前查找文档， 有时候，有人已经回答了与你遇到同样问题的人。文档在 http://www.php.net/support.php 的支持页面上存在。 要订阅 PHP 安装邮件列表，发送一个空的邮件到 php-install-subscribe@lists.php.net. 邮件列表的地址是 php-install@lists.php.net.

如果你想从邮件列表得到帮助，请尽可能精确描述你的环境并给出必要的细节 (哪个操作系统、什么 PHP 版本 、什么 web 服务器， 如果你作为 CGI 或 服务器模块来运行 PHP， 安全模式，等等)，最好还有足够的代码，以便别人能够重现和测试你的问题。

Bug 报告

如果你发现了 PHP 中的 bug，请报告它。除非你报告了这个 bug ， PHP 开发者可能不知道它的存在，这个 bug 便没有被修正的机会。 你能够使用 bug 跟踪系统 http://bugs.php.net/ 来报告 bug 。 请不要发送 bug 报告到邮件列表或个人信箱。 bug 系统也适合用来提交对新特性需求。

在提交任何 bug 报告之前请阅读怎样报告 bug 文档。
