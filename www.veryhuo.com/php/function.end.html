end --  将数组的内部指针指向最后一个单元

说明

mixed end ( array array)

end() 将 array 的内部指针移动到最后一个单元，并返回该单元的值。

例子 1. 简单的 end() 例子

<?php

  $fruits = array('apple','banana','cranberry');

  print end($fruits); // cranberry

?>

参见 current()，each()，next() 和 reset()。
