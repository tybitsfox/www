eregi

(PHP 3, PHP 4 )

eregi -- 不区分大小写的正则表达式匹配

说明

bool eregi ( string pattern, string string [, array regs])

本函数和 ereg() 完全相同，只除了在匹配字母字符时忽略大小写的区别。

例子 1. eregi() 例子

<?php
if (eregi("z", $string)) {
    echo "'$string' contains a 'z' or 'Z'!";
}
?>

参见 ereg()，ereg_replace()，eregi_replace()，stripos() 和 stristr()。
