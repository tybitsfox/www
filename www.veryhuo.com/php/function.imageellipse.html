imageellipse

(PHP 4 >= 4.0.6)

imageellipse -- 画一个椭圆

说明

int imageellipse ( resource image, int cx, int cy, int w, int h, int color)

imageellipse() 在 image 所代表的图像中画一个中心为 cx，cy（图像左上角为 0, 0）的椭圆。w 和 h 分别指定了椭圆的宽度和高度，椭圆的颜色由 color 指定。

注: 本函数添加于 PHP 4.0.6 且需要 GD 2.0.2 或更高版本，可以在 http://www.boutell.com/gd/ 获得。

参见 imagearc()。
