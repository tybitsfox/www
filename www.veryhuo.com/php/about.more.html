如何得到关于 PHP 更多的信息

本手册并不打算提供基本的编程开发指导。如果您是一个初级的程序员，甚至是第一次编程，您可能会发现利用本手册来学习如何用 PHP 编程是一件很困难的事情。为了寻找一本更适合初学者的书，请访问 http://www.php.net/books.php 查看 PHP 相关书籍的列表。

目前，PHP 官方提供了许多邮件列表来讨论 PHP 编程相关的各个方面的问题。当您在 PHP 的开发过程中碰到了难题，而且自己无法找到合适的解决方法时，您可以到该邮件列表寻求帮助。请访问 http://www.php.net/support.php 查看邮件列表清单，我们同时提供了邮件列表的存档和其它在线支持的资料。此外，在 http://www.php.net/links.php 有一个关于 PHP 相关网站的列表，在这些网站中您可以找到和 PHP 相关的文章、论坛和代码。
