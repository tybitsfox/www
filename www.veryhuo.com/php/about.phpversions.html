本手册中所涉及的 PHP 版本

本文档包含有 PHP 4 有关的内容，以及一些关于 PHP 3 移植和兼容的注意事项。PHP 不同版本之间的行为、参数、返回值和其它改变都以“注意”的形式加入了本手册的正文。

您或许会找到一些关于 CVS 版本 PHP（由 CVS 版本控制系统直接得到的最新的开发版本）的文档。如果您本身并不是 PHP 的开发者，并且您并不十分渴望使用 PHP 最新的开发版本，那么标识了“仅 CVS 适用”的章节所描述的特性并不适用于您，或许这些特性会出现在 PHP 下一个稳定版本中。如果您希望下载 CVS 版本，请查阅 CVS 的匿名访问。

您还有可能遇到一些文档仅适用于尚未发布的 PHP 版本（例如 PHP 5.0.0，而当前最新的稳定版本是 4.3.0）。多数情况下，这并不是文档中的错误，我们经常在文档中加入一些目前版本尚未支持的特性描述，但这些特性将在未来某一个确定的 PHP 版本中实现。
