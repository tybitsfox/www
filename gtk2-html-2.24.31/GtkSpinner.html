Style Property Details

The “cycle-duration” style property

  “cycle-duration”           guint

The length of time in milliseconds for the spinner to complete a full loop.

Flags: Read

Allowed values: >= 500

Default value: 1000

Since: 2.20

The “num-steps” style property

  “num-steps”                guint

The number of steps for the spinner to complete a full loop. The animation will complete a full cycle in one second by default (see #GtkSpinner:cycle-duration).

Flags: Read

Allowed values: >= 1

Default value: 12

Since: 2.20
