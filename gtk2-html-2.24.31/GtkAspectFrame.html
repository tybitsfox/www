Property Details

The “obey-child” property

  “obey-child”               gboolean

Force aspect ratio to match that of the frame's child.

Flags: Read / Write

Default value: TRUE

The “ratio” property

  “ratio”                    gfloat

Aspect ratio if obey_child is FALSE.

Flags: Read / Write

Allowed values: [0.0001,10000]

Default value: 1
