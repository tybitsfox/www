The “use-separator” style property

  “use-separator”            gboolean

Whether to put a separator between the message dialog's text and the buttons.

GtkMessageDialog:use-separator has been deprecated since version 2.22 and should not be used in newly-written code.

This style property will be removed in GTK+ 3

Flags: Read

Default value: FALSE
