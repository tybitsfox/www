31.29. sql_implementation_info

表 sql_implementation_info 包含有关 SQL 标准里各种留给具体实现定义的特性的信息。 这些信息主要用在 ODBC 接口的环境里；其它接口的用户可能会觉得这些信息没有什么用处。 出于这个原因，独立的实现信息条目没有在这个描述；你会在 ODBC 接口的描述里找到它们。

Table 31-27. sql_implementation_info 字段

名字数据类型描述
implementation_info_idcharacter_data实现信息条目的标识字串
implementation_info_namecharacter_data实现信息条目的描述性名称
integer_valuecardinal_number 实现信息条目的数值，如果数值在 character_value 字段里包含了， 则为空。
character_valuecharacter_data 实现信息条目的数值，如果该值已经在字段 integer_value 包含， 则为空。
commentscharacter_data可能是一个描述此实现信息条目的注释
