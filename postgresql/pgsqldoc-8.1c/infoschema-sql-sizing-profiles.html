31.33. sql_sizing_profiles

表 sql_sizing_profiles 包含有关 SQL 标准要求的各种剖析文件的 sql_sizing 信息。 PostgreSQL 并不跟踪任何 SQL 剖析文件，所以这个表是空的。

Table 31-31. sql_sizing_profiles 字段

名字数据类型描述
sizing_idcardinal_number尺寸项的标识符
sizing_namecharacter_data尺寸项的描述性名字
profile_idcharacter_data一个剖析文件的标识字串
required_valuecardinal_number SQL 剖析文件对尺寸项要求的数值，如果剖析文件对尺寸项没有限制， 则为 0，如果剖析文件对尺寸项所适用的特性没有任何要求，那么就为空。
commentscharacter_data可能是一个有关该剖析文件里面的尺寸项的注释
