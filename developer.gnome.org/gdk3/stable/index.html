GDK 3 Reference Manual

This document is for the GDK 3 library, version 3.22.15 The latest versions can be found online at https://developer.gnome.org/gdk3/. If you are looking for the older GDK 2 series of libraries, see https://developer.gnome.org/gdk2/.
