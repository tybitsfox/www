gtk_drag_dest_set_track_motion

void gtk_drag_dest_set_track_motion (GtkWidget* widget,
                                     gboolean track_motion);

Tells the widget to emit GtkWidget::drag-motion and GtkWidget::drag-leave events regardless of the targets and the %GTK_DEST_DEFAULT_MOTION flag. This may be used when a widget wants to do generic actions regardless of the targets that the source offers.
