gtk_disable_setlocale

void gtk_disable_setlocale (void);

Prevents gtk_init(), gtk_init_check(), gtk_init_with_args() and gtk_parse_args() from automatically calling <literal>setlocale (LC_ALL, "")</literal>. You would want to use this function if you wanted to set the locale for your program to something other than the user's locale, or if you wanted to set different values for different locale categories. Most programs should not need to call this function.
