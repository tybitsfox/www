gtk_drag_dest_find_target

GdkAtom gtk_drag_dest_find_target (GtkWidget* widget,
                                   GdkDragContext* context,
                                   GtkTargetList* target_list);

Looks for a match between the supported targets of @context and the returning %GDK_NONE. @dest_target_list should usually be the return value from gtk_drag_dest_get_target_list(), but some widgets may have different valid targets for different parts of the widget; in that case, they will have to implement a drag_motion handler that passes the correct target list to this function. and the dest can accept, or %GDK_NONE
