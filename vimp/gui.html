gui

尽管 Vimperator 已经通过 Ex 和普通模式的命令提供了 Firefox 中最常用的功能，访问图形用户界面在某些情况下仍是需要的。Vimperator 的一些命令提供了对系统菜单、标准对话框和侧栏的访问。

:emenu

:emenu {menu}

从命令行访问菜单项 {menu} 。这条命令可以在命令行中访问所有可用的 Firefox 菜单项。参数 {menu} 是一条指向菜单项的层级路径，每级子菜单用句号隔开，例如 :emenu 文件.打开文件… 。

:addo :addons

:addo[ns]

显示浏览器中可用的扩展和主题。用户可以在这个对话框中添加、删除或禁用浏览器中的扩展。注意，不是所有 Firefox 扩展都能和 Vimperator 一起工作，因为 Vimperator 重定义了几乎所有的键位绑定并改变了 Firefox 的外观。

:dia :dialog

:dia[log] [firefox-dialog]

打开一个 Firefox 对话框。

可用的对话框：

about 关于 Mozilla Firefox
addbookmark 将当前页面加为书签
addons 附加组件
bookmarks 书签列表
checkupdates 检查更新
cleardata 清除隐私数据
cookies 显示 Cookie
console 错误控制台
customizetoolbar 定制工具栏
dominspector DOM 查看器
downloads 下载
history 历史记录
import 从其他浏览器导入设置、书签和历史等
openfile 打开文件
pageinfo 页面信息
pagesource 查看页面源代码
places 我的足迹
preferences Firefox 首选项
printpreview 打印预览
printsetup 打印设置
print 打印…
saveframe 帧另存为…
savepage 将页面保存至磁盘
searchengines 管理搜索引擎列表
selectionsource 选中部分的 DOM 源代码

:dl :downl :downloads

:downl[oads]

显示当前下载项的进度。在新标签页中打开原 Firefox 的下载对话框。在这个对话框中用户可以暂停、取消或者恢复下载。

:sbcl :sbclose

:sbcl[ose]

关闭侧栏。

:sbope :sbopen :sb :sbar :sideb :sidebar

:sidebar {name}

打开侧栏。名称 {name} 可以是 Firefox 的 查看→侧栏 菜单下的任意一项。扩展（Add-ons）、首选项（Preferences）和下载（Downloads）对话框也可以在侧栏中打开。
