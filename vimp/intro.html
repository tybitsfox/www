简介

 intro

Vimperator 是 Firefox 浏览器的一个免费扩展。它使 Firefox 的外观和使用方式变得和文本编辑器 Vim 类似。它们有相似的键位绑定，而且你也可以将其称为模式化的浏览器，因为键位的绑定取决于你所处的模式。

warning
警告： 为了提供最 Vim 化的浏览体验， Firefox 的菜单栏和工具栏都被隐藏了。

如果你真的很需要它们，可以输入 :set guioptions+=mT 来把它们找回来。

如果你一点也不喜欢 Vimperator，可以输入 :addons 然后把它卸载或者禁用。

如果你喜欢 Vimperator 但又记不住那些快捷键，可以按 <F1> 或输入 :help 再次打开帮助页面。

特性

 features

  • Vim 式的键位绑定（h，j，k，l，gg，G，0，$，ZZ，<C-f> 等等）

  • Ex 命令（:quit，:open www.foo.com，……）

  • 支持 'wildmode' 选项的 'longest' 模式，支持所有命令的 <Tab> 键补全

  • 使用“选择提示”的浏览模式（使用 f 键打开链接）

  • 对书签和历史 URL 的智能补全（可以搜索标题和 URL）

  • Vim 式的状态栏和 wget 式的进度条

  • 图形界面最小化（可以方便的使用 :set guioptions= 隐藏已经没有用了的的菜单栏和工具栏）

  • 使用 :source 命令导入并执行 Javascript 文件，使用 ~/.vimperatorrc 配置文件（安装 scripts/vimperator.vim 来提供语法高亮）

  • 快速搜索，支持 Firefox 的搜索引擎和书签关键字（:open foo 会在 Google 上搜索 "foo" ，:open ebay terminator 会在 eBay 上搜索 "terminator"）

  • 很多命令都支持指定重复次数（3<C-o> 会自动后退三次）

  • 出错时发出蜂鸣声

  • 支持标记（使用 ma 在当前页面处做上标记 a，使用 'a 返回标记处）

  • 支持快速标记（使用 go{a-zA-Z0-9} 快速打开一个标记过的网页）

  • 支持 :map 与 :command（以及为脚本编写者准备的 feedkeys() 函数）

  • 支持计时命令 :time

  • 支持使用 Vim 式操作在光标模式下移动光标并在可视模式下选择文本

  • 支持外部编辑器

  • 支持使用宏重放键盘操作

  • 支持使用自动命令在特定事件发生时执行代码

  • 一份全面而详细的说明所有命令、映射和选项的帮助文档
