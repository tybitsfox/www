text-search-commands

Vimperator 为 Firefox 自带的文本搜索功能提供了一个类似于 Vim 的界面。这里无法用正则表达式进行搜索，因为 Firefox 不提供本地的正则表达式支持，而且似乎也不准备提供这样的支持。

/

/{pattern}[/]<CR>

正向查找第一次出现 {pattern} 的地方。

如果 "\c" 出现在 {pattern} 的任意位置，模式的匹配会按照 'ignorecase' 选项被打开的情况进行处理。 "\C" 会强制模式在匹配时对大小写敏感。

如果模式中包含 "\l" ，模式的匹配会局限于链接的标题文字，就好像 'linksearch' 选项被打开时一样。 "\L" 会强制模式对整个页面中的内容进行匹配。

?

?{pattern}[?]<CR>

这里 {pattern} 中的修饰符和 / 完全相同。

注意：目前增量搜索仅能用于正向搜索。

n

n

查找下一个匹配。将上次的搜索重复一次。（暂不支持计数 count）

N

N

查找上一个匹配。反向将上次的搜索重复一次。（暂不支持计数 count）

*

*

正向查找光标下的词。

#

#

反向查找光标下的词。

:noh :nohlsearch

:noh[lsearch]

关闭搜索高亮。高亮在下一次搜索或打开 'hlsearch' 选项时会被重新打开。
