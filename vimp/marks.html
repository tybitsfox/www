different-marks marks

Vimperator 支持多种不同的标记：

  • 书签可以让你把最喜欢的网页标记下来以供快速访问。

  • 快速标记允许你标记最多 62 个或组（a-zA-Z0-9）最经常访问的网站。

  • 本地标记可以存储一个页面中的多个位置。

  • 历史记录也是一种特殊的标记，Vimperator 会自动记住你访问过的网站。

书签

 bookmarks

a :bma :bmark

:bma[rk][!] [-title=title] [-keyword=kw] [-tags=tag1,tag2] [url]

a

添加一个书签。

选项说明：

  • -title="自定义标题"（简写：-t）

  • -tags=逗号,分隔的,标签,列表（简写：-T）

  • -keyword=关键字（简写：-k）

使用 [!] 时，总是添加一个新书签。否则第一个和 [url] 相匹配的书签会被更新。

创建一个新书签时，如果省略 [-title]，标题将会被设为页面的标题或是 URL。你可以使用 :bmark 命令并省略 [url] 参数，将当前页面添加为书签，书签的标题默认为页面的标题且没有标签。

A

A

切换当前页面的书签状态。如果当前页面已经被添加为书签，则将其从书签中删除，否则将其添加为书签。和 :bmark 命令相反，该书签仅会被 加星标 ，即添加到 未排序书签 中，而不是书签菜单中。

:bmarks

:bmarks[!] [filter]

列出或打开多个书签。打开屏幕底部的消息窗口，列出所有标题或 URL 和 [filter] 相符的书签。

:bmarks! 和 :bmarks 相似，不同之处在于会在新标签页中打开所有结果。

过滤器也可以包含下列选项： -tags=逗号,隔开的,标签,列表（简写：-T） -max=N（简写：-m）

:delbm :delbmarks

:delbm[arks] [url]

删除一个书签。删除 所有 和 [url] 相匹配的书签。如果省略参数，[url] 的缺省值为当前缓冲区的 URL。使用 <Tab> 键来补全你想要删除的 URL 。

未来版本会支持以下参数：

  • [!] 特别版本，直接删除所有书签！

历史记录

 history

<C-o>

[count]<C-o>

在跳转列表中向后退一位。目前跳转列表就是浏览历史。

<C-i>

[count]<C-i>

在跳转列表中向前进一位。目前跳转列表就是浏览历史。

<M-Left> <A-Left> H

[count]H

后退。如果指定了 [count] 则后退 [count] 个页面。

<M-Right> <A-Right> L

[count]L

前进。如果指定了 [count] 则前进 [count] 个页面。

:ba :back

:[count]ba[ck][!] [url]

后退。如果指定了 [count] 则后退 [count] 个页面。
特别版本 :back! 会直接回退到浏览历史的开始。

:fw :fo :forward

:[count]fo[rward][!] [url]

前进。如果指定了 [count] 则前进 [count] 个页面。

特别版本 :forward! 会直接前进到浏览历史的末尾。

:hs :hist :history

:hist[ory][!] [filter]

显示最近访问过的 URL 。打开屏幕底部的消息窗口，列出所有标题或 URL 和 [filter] 匹配的历史站点。

特别版本 :history! 和 :history 相似，不同之处在于会在新标签页中打开所有结果。

过滤器也可以包含下列选项： -max=N（简写：-m）

快速标记

 quickmarks

go

go{a-zA-Z0-9}

跳转到当前标签页中的一个快速标记。打开当前标签页中的任何快速标记。你可以用 M{a-zA-Z0-9} 标记任何 URL 。这些快速标记是持久的，和浏览器会话无关。

gn

gn{a-zA-Z0-9}

在新标签页中跳转到快速标记。和 go{a-zA-Z0-9} 相似，但会在新标签页中打开快速标记。新标签页是否会被激活取决与 'activate' 选项。

M

M{a-zA-Z0-9}

新增当前 URL 为快速标记。用户可以用 go{a-zA-Z0-9} 在当前标签页中打开被标记过的 URL ，或用 gn{a-zA-Z0-9} 在新标签页中打开被标记过的 URL 。这些快速标记是持久的，和浏览器会话无关。

:delqm :delqmarks

:delqm[arks] {marks}

:delqm[arks]!

删除由列表指定的快速标记。

例如：

  • :delqmarks Aa b p 会删除快速标记 A、a、b 和 p

  • :delqmarks b-p 会删除所有 b 和 p 之间的快速标记

  • :delqmarks! 会删除所有快速标记

:qma :qmark

:qma[rk] {a-zA-Z0-9} [url]

新增 URL 为快速标记。你也可以像这样标记整组页面：
:qmark f http://forum1.com, http://forum2.com, imdb some artist

:qmarks

:qmarks [arg]

显示所有快速标记。如果指定了参数 [arg] 则只列出匹配的快速标记。

本地标记

 localmarks

m

m{a-zA-Z}

标记当前光标的位置。标记 a-z 仅在当前缓冲区内可用而 A-Z 在其他缓冲区中也可用。

` '

'{a-zA-Z}

跳转到当前缓冲区中指定标记的位置。标记 a-z 仅在当前缓冲区内可用而 A-Z 在其他缓冲区中也可用。

:delm :delmarks

:delm[arks] {marks}

:delm[arks]!

删除列表指定的标记。

例如：

  • :delmarks Aa b p 会删除标记 A、a、b 和 p

  • :delmarks b-p 会删除所有 b 和 p 之间的标记

  • :delmarks! 会删除当前缓冲区中的所有标记

:ma :mark

:ma[rk] {a-zA-Z}

标记页面的当前位置。

:marks

:marks [arg]

显示当前页面中的所有位置标记。如果指定了参数 [arg] 则只列出匹配的位置标记。
