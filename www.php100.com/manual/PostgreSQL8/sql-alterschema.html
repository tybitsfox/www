ALTER SCHEMA

Name

ALTER SCHEMA -- 修改一个模式的定义

Synopsis

ALTER SCHEMA name RENAME TO newname
ALTER SCHEMA name OWNER TO newowner

描述

ALTER SCHEMA 修改一个模式的定义。

要使用 ALTER SCHEMA，你必须拥有该模式。 要给一个模式重命名，你必须在该数据库上又有CREATE 权限。 要修改所有者，你必须还是直接或者间接的新的所有角色的成员， 并且必须在该数据库上有 CREATE 权限。 （注意，超级用户自动拥有上面全部权限。）

参数

name

现有模式的名字

newname

模式的新名字，新名字不能以 pg_ 开头， 因为这些名字是保留给系统模式的。

newowner

模式的新所有者。

兼容性

在 SQL 标准里没有 ALTER SCHEMA 语句。

又见

CREATE SCHEMA, DROP SCHEMA
