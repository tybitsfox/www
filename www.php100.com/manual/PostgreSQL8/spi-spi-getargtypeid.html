SPI_getargtypeid

Name

SPI_getargtypeid -- 返回 SPI_prepare 准备的已准备好规划的指定参数的 typeid

Synopsis

Oid SPI_getargtypeid(void * plan, int argIndex)

描述

SPI_getargtypeid 返回 SPI_prepare 准备的已准备好规划的第 argIndex 个参数的类型 ID，用 OID 表示。第一个参数的索引为零。

参数

void * plan

执行规划（SPI_prepare 返回）。

int argIndex

以零为基的参数索引

返回值

给出索引位置的参数的类型 id，如果 plan 为 NULL 或者 argIndex 小于 0 或者大于或等于为 plan 声明的参数个数，则返回 SPI_ERROR_ARGUMENT。
