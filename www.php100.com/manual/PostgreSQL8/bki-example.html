51.4. 例子

下面的命令集将创建名为 test_table，有两个字段 cola 和 colb，类型分别为 int4 和 text 的表，表的 OID 为 420， 然后向该表插入两行。 

create test_table 420 (cola = int4, colb = text)
open test_table
insert OID=421 ( 1 "value1" )
insert OID=422 ( 2 _null_ )
close test_table
