SPI_is_cursor_plan

Name

SPI_is_cursor_plan -- 如果一个 SPI_prepare 准备的规划可以和 SPI_cursor_open 一起使用，则返回真

Synopsis

bool SPI_is_cursor_plan(void * plan)

描述

如果一个 SPI_prepare 准备的规划可以作为参数传递给 SPI_cursor_open 则 SPI_is_cursor_plan 返回真，如果不是这样则返回 false。 评判的标准室这个 plan 代表一个单个命令，并且这个命令是一个没有 INTO 子句的 SELECT。

参数

void * plan

执行规划（SPI_prepare 返回）

返回值

表明该 plan 是否可以生成一个游标的 true 或 false， 如果 plan 是 NULL 则返回 SPI_ERROR_ARGUMENT。
