DROP LANGUAGE

Name

DROP LANGUAGE -- 删除一个过程语言

Synopsis

DROP [ PROCEDURAL ] LANGUAGE name [ CASCADE | RESTRICT ]

描述

DROP PROCEDURAL LANGUAGE 将删除曾注册过的过程语言 name。

参数

name

现存语言的名称。出于向下兼容的考虑，这个名字可以用单引号包围。

CASCADE

自动删除依赖于改语言的对象（比如该语言写的函数）。

RESTRICT

如果存在依赖对象，则拒绝删除。这个是缺省。

例子

下面命令删除 plsample 语言： 

DROP PROCEDURAL LANGUAGE 'plsample';

兼容性

在 SQL 标准里没有 DROP PROCEDURAL LANGUAGE。

又见

ALTER LANGUAGE, CREATE LANGUAGE, droplang
