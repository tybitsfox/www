COMMIT PREPARED

Name

COMMIT PREPARED -- 提交一个早先为两阶段提交准备好的事务

Synopsis

COMMIT PREPARED transaction_id

描述

COMMIT PREPARED 提交一个处于准备好状态的事务。

参数

transaction_id

要提交的事务的标识符。

注意

要提交一个准备好的事务，你必须要么是最初执行该事务的用户，要么是超级用户。 不过你不必在同一个会话里执行该命令。

这条命令不能在事务块里执行。准备好的事务立即提交。

所有目前可用的准备好事务都在系统视图 pg_prepared_xacts 里列出。

例子

提交事务标识符 foobar 标识的事务： 

COMMIT PREPARED 'foobar';

又见

PREPARE TRANSACTION, ROLLBACK PREPARED
