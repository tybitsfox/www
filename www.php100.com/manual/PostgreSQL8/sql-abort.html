ABORT

Name

ABORT -- 退出当前事务

Synopsis

ABORT [ WORK | TRANSACTION ]

描述

ABORT 回卷当前事务并且废弃所有当前事务中做的更新。 这个命令和 SQL 命令 ROLLBACK 的行为完全一样， 只是由于历史原因而保留下来。

参数

WORK
TRANSACTION

可选的关键字，它们没有什么影响。

注意

使用 COMMIT 成功地中止一个事务。

如果不在事务内部发出 ABORT 不会有问题，但是会产生一个警告信息。

例子

退出全部变更： 

ABORT;

兼容性

此命令是 PostgreSQL 基于历史原因做的扩展。 ROLLBACK 是 SQL 中等价的命令。

又见

BEGIN, COMMIT, ROLLBACK
