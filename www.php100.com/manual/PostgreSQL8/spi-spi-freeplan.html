SPI_freeplan

Name

SPI_freeplan -- 释放一个前面保存的规划

Synopsis

int SPI_freeplan(void *plan)

描述

SPI_freeplan 释放一个以前由 SPI_prepare 返回的或者是用 SPI_saveplan 保存的命令执行规划。

参数

void * plan

指向要释放的规划的指针

返回值

如果 plan 为 NULL，返回 SPI_ERROR_ARGUMENT。
