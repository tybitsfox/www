SPI_exec

Name

SPI_exec -- 执行一个读/写命令

Synopsis

int SPI_exec(const char * command, long count)

描述

SPI_exec 和 SPI_execute 一样，只是相当于后者的 read_only 参数总是 false。

参数

const char * command

包含需要执行的命令的字串

long count

处理或者返回的最大行数

返回值

参阅 SPI_execute。
