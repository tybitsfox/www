DROP VIEW

Name

DROP VIEW -- 删除一个视图

Synopsis

DROP VIEW name [, ...] [ CASCADE | RESTRICT ]

描述

DROP VIEW从数据库中删除一个现存的视图。 执行这条命令必须是视图的所有者。

参数

name

要删除的视图名称（可以有模式修饰）。

CASCADE

自动删除依赖此视图的对象（比如其它视图）。

RESTRICT

如果有依赖对象存在，则拒绝删除此视图。这个是缺省。

例子

下面命令将删除视图 kinds： 

DROP VIEW kinds;

兼容性

这条命令遵循 SQL 标准。只是标准只允许一条命令删除一个视图。
