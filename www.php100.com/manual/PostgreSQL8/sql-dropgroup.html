DROP GROUP

Name

DROP GROUP -- 删除一个用户组

Synopsis

DROP GROUP name

描述

DROP GROUP 从数据库中删除指定的组。组中的用户不被删除。 组中的用户不被删除。

参数

name

现存组名。

注意

删除一个在对象上赋予了权限的组是不明智的。 目前，这一点还不是强制的，但是很有可能在未来的 PostgreSQL 版本众将检查这个错误。

例子

删除一个组： 

DROP GROUP staff;

兼容性

SQL 标准里没有 DROP GROUP。

又见

ALTER GROUP, CREATE GROUP
