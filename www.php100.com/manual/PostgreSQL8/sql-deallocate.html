DEALLOCATE

Name

DEALLOCATE -- 删除一个准备好的查询

Synopsis

DEALLOCATE [ PREPARE ] plan_name

描述

DEALLOCATE 用于删除前面准备好的查询。 如果你没有明确 DEALLOCATE 一个准备好的查询， 那么它在会话结束的时候删除。

有关已准备好的查询的更多信息。参阅 PREPARE。

参数

PREPARE

这个关键字被忽略。

plan_name

要删除的已准备查询。

兼容性

SQL 包括一个 DEALLOCATE 语句，但它只是用于嵌入的 SQL 客户端。

又见

EXECUTE, PREPARE
