SPI_getbinval

Name

SPI_getbinval -- 返回声明字段的二进制值

Synopsis

Datum SPI_getbinval(HeapTuple row, TupleDesc rowdesc, int colnumber, bool * isnull)

描述

SPI_getbinval 以数据的内部形式（像类型 Datum）返回声明字段的数值。

这个函数并不为数据分配新的空间。 如果是传递引用，那么返回值将是指向传递的行的指针。

参数

HeapTuple row

输入的要检查的行

TupleDesc rowdesc

输入行描述

int colnumber

字段编号（从 1 开始记）

bool * isnull

字段里空值的标志

返回值

返回该字段的二进制值。如果字段为空，那么 isnull 指向的变量设置为真，否则为假。

错误时，SPI_result 设置为 SPI_ERROR_NOATTRIBUTE。
