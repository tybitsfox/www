SPI_gettypeid

Name

SPI_gettypeid -- 返回声明的字段的数据类型 OID

Synopsis

Oid SPI_gettypeid(TupleDesc rowdesc, int colnumber)

描述

SPI_gettypeid 返回声明字段的数据类型的 OID。

参数

TupleDesc rowdesc

输入行描述

int colnumber

字段编号（从 1 开始记）

返回值

声明字段的数据类型的 OID，或者错误时是 InvalidOid。 在出错的时候，SPI_result 设置为 SPI_ERROR_NOATTRIBUTE。
