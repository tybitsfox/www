RESET

Name

RESET -- 把一个运行时参数值恢复为缺省值

Synopsis

RESET name
RESET ALL

描述

RESET 将运行时参数恢复为缺省值。 RESET 是下面语句的一个变种 

SET parameter TO DEFAULT

请参考 SET 命令令获取允许的变量值和缺省值的详细信息。

缺省值是定义成变量可以拥有的数值，并且在当前会话中没有用 SET 设置过。这个数值的实际源头可能是编译的缺省， postmaster 的配置文件，或者是命令行开关，或者是每数据库或每用户的缺省设置。 参阅 Chapter 17 获取细节。

参阅 SET 手册页获取有关RESET 的事务行为的细节。

参数

name

请参考 SET 语句获取有关可用的参数的详细说明。

ALL

把所有运行时参数设置为缺省值。

例子

把 geqo 配置变量设为其缺省值： 

   
RESET geqo;

兼容性

RESET 是 PostgreSQL 扩展。
