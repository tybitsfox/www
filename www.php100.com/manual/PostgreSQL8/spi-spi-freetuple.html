SPI_freetuple

Name

SPI_freetuple -- 释放在上层执行者环境里分配的一行

Synopsis

void SPI_freetuple(HeapTuple row)

描述

SPI_freetuple 释放一个前面在上层执行者环境里分配的行。

这个函数和 heap_freetuple 没什么区别。保留它主要是为了向后兼容现有代码。

参数

HeapTuple row

要释放的行
