SPI_copytuple

Name

SPI_copytuple -- 在上层执行者环境里制作一个行的拷贝

Synopsis

HeapTuple SPI_copytuple(HeapTuple row)

描述

SPI_copytuple 在上层执行者环境里制作一个行的拷贝。 这个函数通常用于从触发器里返回一个修改过的行。在声明为返回复合类型的函数里， 应该使用 SPI_returntuple。

参数

HeapTuple row

要拷贝的行

返回值

拷贝行；只有在 tuple 是 NULL 的时候为 NULL
