31.40. view_table_usage

视图 view_table_usage 标识所有在一个视图的查询表达式中 （定义视图的 SELECT 语句）使用的表的名字。只有在当前用户是这个表的所有者的时候才会包含这个表。

注意: 没有包括系统表。这个在将来应该修补。

Table 31-38. view_table_usage 字段

名字数据类型描述
view_catalogsql_identifier包含该视图的数据库的名字（总是当前数据库）
view_schemasql_identifier包含该视图的模式的名字
view_namesql_identifier视图的名字
table_catalogsql_identifier 包含被视图使用的表的数据库的名字（总是当前数据库）
table_schemasql_identifier 包含被视图使用的表的模式的名字
table_namesql_identifier 视图使用的表的名字
